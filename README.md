# Timebotpy Artificial Intelligence Framework
This is a Artificial Intelligence Framework that use to create a group of AI entity 
and corporate with each other.

## Language and Environment Requirements
This software is written in python 2.7.x and based on Google Cloud Platform.

## Missions
Timebotpy acts as a base class to extends, 
and extended classes work as a group to accomplish a goal.

## Philosophies
The whole framework designed in Tree Structure. Timebotpy and it's extends classes works with each other in 
Fractal manner.
