# Timebotpy Artificial Intelligence Framework

This is a Artificial Intelligence Framework that use to create a group of AI entity and corporate with each other.

## Language and Environment Requirements
This software is written in python 2.7.x and based on Google Cloud Platform.

## Missions
Timebotpy is based on Symbolic Artificial Intelligence. It's mission is to convert meanings or commands into a format of though, and been able to convert into other forms of expression, for example convert a command to a change of website.

Timebotpy acts as a base class to extends, and extended classes work as a group to accomplish a goal.
